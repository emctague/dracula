# Dracula

This is a simple counter program, available at
[emctague.github.io/dracula](http://emctague.github.io/dracula).

## Keybindings

The keybindings are not documented on the webpage, but still exist:

|Action|Keys|
|------|----|
|Decrement|`down`, `enter`, `space`, `-`|
|Increment|`up`, `+`, `=`|
|Reset|`r`, `escape`|
|Start|`enter`|
